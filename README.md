# 图书管理系统简介 java312

## 项目概述

本系统是基于Springboot和Vue的前后端分离的图书管理系统。它使用了当前流行的技术栈进行开发，旨在实现高效、易用的图书管理功能。

## 技术栈

### 后端
- JDK 1.8
- Maven
- MySQL（建议5.0及以上版本）
- SpringBoot 2
- MyBatis
- Redis

### 前端
- Vue 2 全家桶
- Element-UI
- Axios

## 功能模块

系统分为以下两个角色：

### 管理员
- 图书信息管理
- 图书类型管理
- 图书借阅管理
- 用户管理
- 修改密码

### 普通用户
- 图书信息查看
- 图书借阅
- 修改密码

## 系统角色

系统定义了两种角色，分别为管理员和普通用户，以满足不同用户群体的需求。

## 运行环境

- 后端：Java 1.8环境，兼容主流操作系统
- 前端：现代浏览器环境
